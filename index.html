虚拟显示器——在没有显示器连接时激活Windows 10多显示器设置

2021-10-05 #virtual display #虚拟显示器
原文地址：Activating a Secondary Display on Windows 10 when no Monitor is Connected 下面为机翻，调整了关键的命令行关键字，以便复制运行。需要注意的是本文是虚拟的显示器，不是虚拟桌面。该功能可以将虚拟显示器扩展投屏分享，不与主屏发生联系，且不用外接连接线。 在某些情况下，您需要模拟一个或多个连接到您的系统的辅助显示器。这是最需要的远程控制应用程序或USB类型的显示器，如Amyuni USB移动显示器产品。 在Windows 10之前，很容易通过显示设置控制面板激活VGA显示器来模拟连接第二个显示器。如果你正在寻找旧系统的解决方案，滚动到这篇文章的结尾。 Windows 10删除了在没有连接VGA显示器时激活VGA显示器的选项。我们在这里提供了一个简单而免费的解决方案，可以在您的系统上创建多达4个虚拟显示器。 以下是将虚拟显示器添加到您的系统的过程(仅限Windows 10 !) - 从https://www.amyuni.com/downloads/usbmmidd_v2.zip 下载我们的虚拟显示驱动程序&gt; - 将zip文件解压缩到一个空文件夹，例如c: \temp\usbmmidd_v2 - 请确保您像阅读任何其他软件产品一样阅读License.txt文件 - 以管理员身份打开命令提示符窗口(除非以管理员身份运行，否则不能向系统添加设备) - 执行如下命令: - CD c:\temp\usbmmid_v2(或任何你选择的目标文件夹) - deviceinstaller64 install usbmmidd.inf usbmmidd - 请确保您看到驱动程序由Amyuni Technologies Inc.签署的消息。这是一个确认，驱动程序通过微软驱动程序签名程序，是无病毒的 - deviceinstaller64 enableidd 1 为了在Windows 10中添加4个虚拟显示器，您最多可以运行最后一条命令4次。 如果你使用的是32位系统，将&quot;deviceinstaller64&quot;替换为&quot;deviceinstaller&quot; 包含的批处理文件usbmmid .bat会自动运行这些命令，并在32位或64位版本的设备安装程序之间进行选择。如果您不习惯打开命令提示符并输入这些命令，右键单击usbmmid .bat和“以管理员身份运行”。 就是这样。现在在显示设置中有一个高清晰度显示器。默认分辨率是1920x1080像素，如果您需要其他分辨率，请检查包含的idd_instructions.txt文件，以了解如何指定您自己的分辨率。在你的系统的显示设置中，你可以放置显示器，改变分辨率，扩展桌面或者做任何你用真实显示器做的事情。 虚拟显示器默认支持以下显示分辨率： 1024x768像素 1360 x 768像素 1440 x 900像素 1600 x 900像素 1600 x 1200像素 1920 x 1080像素(默认分辨率) 1920 x 1200像素 2560 x 1440像素 3840x2160像素 执行上述第一步后，您可以通过更改以下注册表项来修改支持的分辨率列表： HKEY\U LOCAL\U MACHINE\SOFTWARE\Microsoft\Windows NT\CurrentVersion\WUDF\Services\usbmmIdd\Parameters\Monitors 通过此注册表项最多可以定义10个分辨率。键的名称应该从“0”到“9”。（默认）注册表值定义默认显示器分辨率。 禁用虚拟显示器。 使用实例 deviceinstaller64 enableidd 0 (如果添加了多个虚拟显示，请多次运行此命令) 重新激活: deviceinstaller64 enableidd 1 要从你的系统中完全移除驱动程序，你可以通过设备管理器卸载“USB移动显示器虚拟显示”，或者运行以下命令: deviceinstaller64 stop usbmmidd deviceinstaller64 remove usbmmid Windows 10之前的操作步骤如下: - 右键单击桌面，选择“屏幕分辨率”或“显示设置” - 点击主显示器旁边的“检测” - 选择“未检测到另一个显示”-&gt;多个显示-&gt;尝试连接VGA - 这将添加第二个虚拟显示器，然后您可以将其配置为真实显示器 其他适用于所有系统的解决方案是购买一个硬件狗连接到你的系统的HDMI、DVI或VGA连接器，以模拟显示器是实际连接的。 欲了解更多有关USB移动显示器的详细信息，请访问我们的专门网站: www.usbmobilemonitor.com Amyuni开发团队
